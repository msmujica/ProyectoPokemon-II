Detailed Description

Clase que representa el efecto de "envenenar" a un Pokémon. Un Pokémon envenenado pierde una cierta cantidad de vida por cada turno que pasa.

bool Library.EfectoEnvenenar.ProcesarEfecto ( Pokemon pokemon)
inline

Procesa el efecto de "envenenar" durante cada turno del Pokémon afectado. Cada vez que se llama, el Pokémon pierde un porcentaje de su vida debido al veneno.

Parameters
pokemonEl Pokémon que está bajo el efecto de veneno.
Returns
true si el efecto sigue activo (es decir, el Pokémon no ha muerto debido al veneno). false si el efecto ha terminado (es decir, el Pokémon ha quedado fuera de combate por el veneno).
